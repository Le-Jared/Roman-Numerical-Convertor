# Roman-Numerical-Convertor
Simple web app for assessment with testing

# How to use on macOS/Linux
Create the Environment
$ mkdir myproject
$ cd myproject
$ python3 -m venv venv

Activate the Environment
$ . venv/bin/activate

Install Flask
$ pip install Flask

# How to use on Windows
Create the Environment
> mkdir myproject
> cd myproject
> py -3 -m venv venv

Activate the Environment
> venv\Scripts\activate

Install Flask
$ pip install Flask
